# GeniusTag Landing Page (lp-003)

## Overview
Conversion-optimized landing page for GeniusTag Smart Pet ID Tag.

## Source Template
Based on Petivity GPS Dog Tracker product page (plain-landing-page-swipe-002).
**This is a RESKIN, not a redesign** - preserves exact HTML structure while replacing brand elements.

## Copywriting Frameworks Applied
- Eugene Schwartz: Market awareness matching
- Gary Bencivenga: Persuasion equation (Proof x Promise + Emotional Resonance)
- Clayton Makepeace: Emotional dominance (Fear > Amplify > Channel > Resolve)
- Dan Kennedy: Magnetic marketing (bleeding neck problem)
- Frank Kern: Intent-based branding (Results in Advance)

## Reskin Methodology
The following elements were replaced while preserving the original structure:
- **Brand Name**: Petivity → GeniusPet
- **Product Name**: GPS + Activity Tracker → Smart Pet ID Tag
- **Colors**: Teal (#0d7978) → Red (#D74242)
- **Typography**: Added Gilroy font family
- **Copy**: Adapted from GPS tracking to NFC identification

## GeniusPet Brand Elements
- Primary Color: #D74242 (Red)
- Secondary Color: #C43C3C (Dark Red)
- Text Color: #2B2B2B (Black)
- Background: #FBF8F4 (Beige)
- Font: Gilroy (various weights)

## Key Features
- Self-contained HTML (all assets inlined)
- GeniusPet brand colors and Gilroy typography
- Mobile-responsive design
- FAQ accordion functionality
- Smooth scroll navigation
